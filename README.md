# Instacraper

> Scrap images from an instagram url and save them locally

## Installation

```sh
npm install -g instacraper
```

## Usage

```sh
instacraper 'https://www.instagram.com/p/Bx3F46OFTju/'
```

Show help with `-h`

```sh
instacraper -h
```
